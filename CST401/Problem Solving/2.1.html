## Solving Problems by Searching: Problem-Solving Agents Problem-solving agents are a type of goal-based agent. They utilize atomic representations, meaning the states of the world are treated as wholes without internal structure visible to the problem-solving algorithms (Ihsana Muhammed, CST401). Goal-based agents employing more advanced factored or structured representations are typically called planning agents. Goals are crucial for organizing behavior by limiting the objectives and actions the agent needs to consider. **Steps Required to Solve a Problem:** 1. **Goal Formulation:** This involves defining a target or goal requiring actions to achieve, based on AI agent capabilities. It's formulated as a set of world states where the goal is satisfied (Ihsana Muhammed, CST401). 2. **Problem Formulation:** This determines the actions needed to achieve the formulated goal. 3. **Action Selection:** Reaching the goal state from the initial state requires actions—operators causing transitions between world states. These actions should be abstract enough to avoid excessive detail and uncertainty, for example, "turn left" instead of "turn left 30 degrees" (Ihsana Muhammed, CST401). **Searching for Solutions:** The course will cover both uninformed and informed search strategies. **2.4 Uninformed Search Strategies:** * **Breadth-First Search (BFS)** * **Uniform-Cost Search** * **Depth-First Search (DFS)** * **Depth-Limited Search** * **Iterative Deepening Depth-First Search** **2.5 Informed Search Strategies:** * **Best-First Search** * **A\* Search** **2.6 Heuristics:** (Further details on heuristics would be included here, based on additional information from the provided text, if available). **Atomic Representation:** In an atomic representation, each state of the world is indivisible, lacking internal structure (Ihsana Muhammed, CST401). An example is finding a driving route across a country; the state could be simplified to the current city—a single atom of knowledge (Ihsana Muhammed, CST401).
