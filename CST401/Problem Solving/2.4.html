**Uninformed Search Strategies** Uninformed search, also known as blind search, does not utilize domain-specific knowledge such as the goal's location or closeness. It operates using brute force, focusing solely on traversing the search tree and identifying leaf and goal nodes. The search tree is explored without information about the search space, including the initial state, operators, and goal test. Each node is examined until the goal node is reached. Five main types of uninformed search exist: * **Breadth-First Search (BFS):** BFS is a common strategy for traversing trees or graphs. It searches breadthwise, starting from the root node and expanding all successor nodes at the current level before moving to the next level. BFS is implemented using a FIFO (First-In, First-Out) queue data structure. * **Advantages:** BFS guarantees finding a solution if one exists. If multiple solutions exist, it finds the minimal solution (requiring the fewest steps). * **Disadvantages:** BFS requires significant memory as each tree level must be stored to expand the next.
